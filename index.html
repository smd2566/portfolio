Alien Escape

"Alien Escape" is a fast paced game created in Unity where you control a vehicle to avoid alien light beams. The game becomes more difficult as you progress, with more UFOs spawning each level. Every 10 levels, the number of UFOs reset and are then sped up, and the process keeps repeating until a game over.

Carnival Builder

Carnival Builder is a rich web application that utilizes a backend API in order to communicate with a server that stores various types of data. The data is retrieved through GET/HEAD/POST requests and is served to the client in JSON format. The application allows the user to add information in order to build a carnival to their liking.

Par for the Curse

"Par for the Curse" is a visual novel / golf hybrid game that revolves around solving a mystery by playing golf matches to gather clues. The player has to meet the par for each course in order to advance the story. Each level uses physics and logic to simulate an arcade style of minigolf. I served as the lead level designer/modeler within the development team for this project, and contributed to general overhead/design decisions.

Asteroids

"Asteroids" is a shooter game in which you control a spaceship with the goal of destroying asteroids to gain points. Each asteroid is programmed with velocity and acceleration vectors, and split into a smaller asteroid upon being destroyed. Collision detection methods are also implemented to create interaction between game elements. This game was programmed in 2D Unity and utilizes random algorithms along with vector math.
